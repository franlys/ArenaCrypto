# Skill: ui-ux-pro-max

**Objetivo**: User experience de alto nivel con paletas vibrantes tipo eSports y tipografías impactantes.

## Paleta de Colores ArenaCrypto
- **neon-cyan**: `#00F5FF` — Acción principal, botones, estados activos.
- **neon-purple**: `#8B5CF6` — Secundario, badges, elementos decorativos.
- **gold**: `#FFD700` — Usuarios VIP, Logros, Ranks #1.
- **dark-bg**: `#09090B` — Fondo principal profundo.
- **dark-card**: `#12121A` — Paneles, chats y modales.

## Tipografía
- **Orbitron**: Headers primarios y marca (Máximo impacto).
- **Rajdhani**: Títulos secundarios y nombres de juegos.
- **Inter**: Interfaz general y textos de cuerpo.

## UX Patterns
- **Feedback Inmediato**: Cada click o acción debe tener una reacción visual instantánea.
- **Skeletons**: Uso de skeletons para estados de carga, evitando "flashes" de blanco o vacío.
- **Confirmaciones**: Acciones de alto riesgo (como realizar una apuesta grande) deben requerir confirmación explícita.
- **Ads No Intrusivos**: Zonas periféricas claras que respetan el flujo central del jugador.
