# Skill: frontend-design

**Objetivo**: Interfaces premium, no genéricas, diseñadas para la élite de los eSports.

## Principios
- **No Templates**: Cada pantalla debe sentirse como un producto de alta calidad hecho a medida.
- **Jerarquía Visual**: Uso de contraste y espacios para guiar al usuario a las acciones críticas (Betting, Matching).
- **Consistencia**: Mantener el ADN de la marca ArenaCrypto en cada botón, modal y transición.
- **Dark-First**: Optimizado para pantallas OLED y ambientes de gaming de baja luz.

## Estándares para ArenaCrypto
- **Leaderboard Estilo Stadium**: Debe impresionar visualmente con efectos de profundidad y neón.
- **Dashboard Funcional**: Organización limpia de fondos, suscripciones y partidas activas.
- **Formularios con Feedback**: Validaciones visuales inmediatas con colores neón para estados de éxito/error.
